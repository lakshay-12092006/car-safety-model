🚗 Car Safety Prediction System

The Car Safety Prediction System is a Machine Learning project that predicts the safety category of a car using various vehicle features such as buying price, maintenance cost, number of doors, seating capacity, luggage boot size, and estimated safety level.

The project uses a trained classification model to analyze the input parameters and classify a car into one of the predefined safety categories, enabling users to make informed decisions while comparing vehicles.

✨ Features
Predicts car safety category using Machine Learning
User-friendly web interface
Fast and accurate predictions
Real-time input validation
Responsive and easy-to-use design
🛠️ Technologies Used
Python
Scikit-learn
Pandas
NumPy
Joblib
Gradio (or Streamlit, if applicable)
📌 Input Parameters
Buying Price
Maintenance Cost
Number of Doors
Seating Capacity
Luggage Boot Size
Safety Rating
🎯 Output
Predicted Car Safety Category (e.g., Unacceptable, Acceptable, Good, or Very Good)
