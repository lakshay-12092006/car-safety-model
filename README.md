# 🚗 Car Safety Prediction System

An AI-powered Machine Learning web application that predicts the **safety category of a car** based on its specifications. The model analyzes different vehicle attributes and classifies the car into one of the predefined safety categories, helping users make informed decisions while comparing vehicles.

## ✨ Features

- 🚗 Predicts car safety category using Machine Learning
- ⚡ Instant prediction results
- 🖥️ Simple and user-friendly web interface
- 📊 Accurate classification based on vehicle specifications
- 📱 Responsive design
- 🌐 Easy to deploy on Render or other cloud platforms

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Joblib
- Gradio

---

## 📥 Input Parameters

The model takes the following inputs:

- Buying Price
- Maintenance Cost
- Number of Doors
- Seating Capacity (Persons)
- Luggage Boot Size
- Safety Rating

---

## 📤 Output

The system predicts one of the following car safety categories:

- Unacceptable (unacc)
- Acceptable (acc)
- Good (good)
- Very Good (vgood)
