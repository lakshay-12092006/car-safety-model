# 🏦 Loan Prediction System

A Machine Learning-based **Loan Prediction System** that predicts whether a loan application is likely to be **Approved** or **Rejected** based on applicant information. The project uses historical loan data to train a classification model and provides an easy-to-use web interface for making predictions.

## 🚀 Live Demo
🔗 https://loan-prediction-model-hz5l.onrender.com

---

## 📌 Features

- Predicts loan approval status instantly
- User-friendly web interface
- Machine Learning classification model
- Data preprocessing and feature encoding
- Fast and accurate predictions
- Responsive design
- Easy to deploy on Render

## 📊 Input Parameters

The model predicts loan approval based on the following details:

- Gender
- Married
- Dependents
- Education
- Self Employed
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Loan-Prediction-System.git
```

### Move into Project Folder

```bash
cd Loan-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Project

```bash
streamlit run app.py
```
